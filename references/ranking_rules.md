# 推荐分级策略

## 等级权重排序（从高到低）

```
世界田联：白金标 > 金标 > 精英标 > 标牌
田协认证：A类 > B类 > C类
```

## 综合推荐优先级

### 必跑（无论距离）
- 💎白金标
- 🥇金标
- 🏅精英标

### 品质推荐（结合距离综合判断）
- 🎖️标牌 + 距用户近
- 🅰️田协A类 + 距用户近

### 可考虑
- 🅰️田协A类（距离适中）
- 🅱️田协B类（距用户近）

### 一般不推荐
- C类（除非极近 <30min 或完赛物资特别好）
- 远距离的标牌级以下赛事

## 赛事级别来源优先级
1. 田协年度目录 → A/B/C类（唯一权威）
2. 世界田联官网/报道 → 标牌等级
3. 闹跑详情页 → 仅作参考，不单独采信
