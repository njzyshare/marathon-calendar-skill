# 推荐分级策略

## 等级权重排序（从高到低）

```
世界田联：白金标 > 金标 > 精英标 > 标牌
田协认证：A类 > B类 > C类
```

## 板块分类规则

### 🎯 已报名
用户已报名的赛事，按状态排序：已中签(✅) > 已报名(⏳) > 未中签(❌)

### 🔜 蹲省外 / 蹲省内
准入条件：**A类及以上**（田协A类/世界田联标牌及以上）
排序：按标牌等级从高到低

### 🏃 其他推荐赛道
准入条件：非A类赛事但有突出特色，例如：
- 报名送跑鞋等特殊福利
- 完赛物资极其丰盛（如义乌小商品之都）
- 风景绝美（如桂林山水）
- 其他跑圈公认的特色

### 一般不推荐
- C类且无突出特色
- 远距离的标牌级以下赛事

## 赛事级别来源优先级
1. 田协年度目录 → A/B/C类（唯一权威，不可单信闹跑页面）
2. 世界田联官网/报道 → 标牌等级
3. 闹跑详情页 → 仅作参考
