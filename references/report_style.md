# 报告书写风格规范

## 整体结构

报告分为以下几个板块，按顺序排列：

1. **标题头** — 用户姓名 + 年份 + 主题
2. **推荐逻辑说明** — 简述推荐原则
3. **已报名板块** — 用户已报名的赛事
4. **省内板块** — 用户所在省份的赛事全览
5. **省外高标板块** — 省外高标等级赛事
6. **赛道推荐板块** — 近期已官宣且有特色的赛事
7. **底部声明** — 更新日期 + 数据来源

## 每条赛事卡片格式

```
[赛事名称] [等级标签1] [等级标签2] [位置标签]
  📍 Meta行：交通耗时(含车次票价) · 起终点路线 · 日期时间
  🏷️ 备注行：赛道特点/爬升/旅游特色/PB潜力
  [状态标签] 报名状态 · 关键日期
```

### 等级标签一览
| 标签 | 含义 | 颜色指向 |
|------|------|---------|
| 💎白金标 | 世界田联最高 | 紫色 |
| 🥇金标 | 世界田联金标 | 金色 |
| 🏅精英标 | 世界田联精英 | 蓝色 |
| 🎖️标牌 | 世界田联标牌 | 靛蓝 |
| 🅰️A类 | 田协A类认证 | 绿色 |
| 🅱️B类 | 田协B类认证 | 紫色 |
| 📌必跑 | 强烈推荐 | 粉色 |
| 🏠超近 | 离家近 | 粉色 |

### 备注行（🏷️）内容规范
- 必须包含：赛道平坦程度 + 累计爬升（如有）
- 推荐包含：沿途景点、城市特色
- 可选包含：历年PB数据、完赛物资、交通福利
- 不超过两行，信息密度优先

### 状态标签一览
| 标签 | 含义 |
|------|------|
| ✔️ 已报名 | 已完成报名 |
| 🔔 等报名 | 尚未开放报名 |
| ⏰ 报名中 | 正在开放报名 |
| ❗ 即将截止 | X天后截止 |
| ❌ 已结束 | 已完赛 |
| 🚫 已取消 | 取消举办 |

## 文字结论风格（对话框输出）

与报告风格一致但更精简，每条赛事控制在3行以内：

```
🏃 [赛事名称] [等级]
  📅 比赛：X月X日 · 📍 [城市] · 🚄 [交通耗时]
  🏷️ [赛道特色一句话]
  💬 [推荐理由或状态说明]
```
