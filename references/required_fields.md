# 必采字段清单

每条赛事必须采集12个关键字段，缺一不可：

| # | 字段 | 获取方式 | 说明 |
|---|------|---------|------|
| 1 | **赛事名称** | 闹跑首页 / 新闻 | |
| 2 | **田协认证级别** (A/B/C) | 田协年度目录 | **唯一权威**，闹跑页面不采信 |
| 3 | **世界田联标牌等级** | 世界田联官网/报道 | 白金标/金标/精英标/标牌 |
| 4 | **比赛日期 (精确到日)** | 闹跑 + 新闻确认 | 如果闹跑只写"12月"必须搜新闻确认 |
| 5 | **报名开始日期** | 新闻 / 赛事官网 | 没有则标注"待定" |
| 6 | **报名截止日期** | 新闻 / 赛事官网 | 没有则标注"待定" |
| 7 | **出签日期** | 新闻 / 赛事官网 | 抽签/先到先得/免抽签需标注 |
| 8 | **赛事项目** (全马/半马) | 闹跑详情页 | 必须标注具体项目 |
| 9 | **距用户交通耗时** | 携程/12306 | 最快高铁车次+耗时+票价 |
| 10 | **累计爬升** | 搜索引擎 / 闹跑 | 跑友手表数据/官方公布数据 |
| 11 | **赛道特色备注** | 搜索引擎 | 爬升+景点+旅游特色 |
| 12 | **撞期检查** | 对比同省同日赛事 | 如有撞期必须标注"⚠️ 撞期" |

## 撞期检查规则
- 采集完成后，按省份分组所有赛事
- 同省同日赛事 → 标注"⚠️ 与XX同天"
- 优先推荐级别高/距离近的
