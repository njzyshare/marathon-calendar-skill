# 每条赛事必须采集的关键字段

每场赛事在报告中展示时，**必须采集以下所有字段**，缺哪个补哪个，不能跳过。

## 必采字段列表

| 序号 | 字段 | 来源 | 说明 |
|------|------|------|------|
| 1 | **赛事名称** | 闹跑首页 / 田协目录 | 完整名称 |
| 2 | **认证级别** | 田协目录(A/B/C) + 世界田联报道(标牌) | 双体系都要查 |
| 3 | **比赛日期** | 闹跑详情页 → 如仅显示月份，**必须搜最新新闻确认具体日期** | ⚠️ 模糊日期不可放过 |
| 4 | **报名开始/截止** | 闹跑详情页 / 最新新闻 | 精确到日 |
| 5 | **出签日期** | 最新新闻 / 赛事官网 | 有抽签的就查 |
| 6 | **距用户所在地交通** | 12306/携程查高铁 | 标注最快车次+耗时+票价 |
| 7 | **赛事项目** | 闹跑详情页 | 全马/半马/其他 |
| 8 | **起终点** | 闹跑详情页 | 起点→终点 |
| 9 | **累计爬升** | 闹跑详情页 / 搜索引擎 | 🟢平坦(<100m) 🟡有坡(100-300m) 🔴爬升大(>300m) |
| 10 | **赛道备注** | 搜索引擎搜"赛事名 赛道 爬升" | 景点、城市特色、PB潜力、完赛物资 |
| 11 | **报名状态** | 综合判断 | 报名中/等报名/已结束/已取消 |
| 12 | **是否撞期** | 对比同省/同日期赛事 | 同一天的多场赛事要标注"撞期" |

## 采集优先级

```
第一优先（从闹跑详情页和搜索引擎同时获取）：
  比赛日期 → 如果只显示月份，必须搜新闻确认具体日
  报名时间 → 闹跑有则用，没有就搜新闻
  出签日期 → 搜新闻查
  累计爬升 → 闹跑有则用，没有就搜"赛事名 赛道 爬升"
  起终点 → 闹跑详情页

第二优先（必须搜新闻确认）：
  比赛具体日期（闹跑只显示月份时）
  报名开始/截止日期（闹跑没有时）
  出签日期
  赛道特色/爬升补充

补充来源：
  搜狐/腾讯新闻 → 赛事定档新闻
  抖音/微信公众号 → 官方报名信息
  马拉松报名网 → 报名状态
```

## 撞期检查规则

- 同一省份的赛事、同一天或同周末的赛事，必须做**撞期标注**
- 标注格式：`⚠️ 与XX赛事同天，建议二选一`
- 优先推荐级别更高的那场
