# 马拉松个人赛历和推荐 Skill

## 核心工作流程

当用户咨询马拉松相关问题时，按以下流程：

### 1. 先回答，再问是否需要报告
- **默认只输出对话框文字结论**，不要自动生成文件/HTML/图片
- 文字结论需包含：比赛日期、报名状态、认证级别、距用户的大致交通耗时
- 文字结论之后，**主动询问用户**："需要我把这些整理成报告吗？你可以选择 HTML 页面或图片长图"

### 2. 报告生成规则
- **生成报告前必须询问用户两个选择**：
  1. 是否要生成报告（默认不生成）
  2. 如果要生成，选择格式：**HTML页面** 或 **图片长图（PNG）**
- 用户选择后再执行生成操作
- 不需要询问用什么工具生成，直接按用户选择的格式生成即可

### 3. 交通耗时查询规范
- **必须从12306/携程等平台查询真实数据**，禁止凭距离估算
- 查询方法：用WebFetch或WebSearch查"福州到XX 高铁 最快"
- 标注具体车次和票价（如"最快2h49min G258 ¥344"）
- 远距离（高铁>6h）建议标注"建议飞机"

## 数据来源与权威性权重

### 第一梯队 - 最权威（田协官方认证级别）
- **中国田协《2026年全国马拉松赛事目录》**（PDF/文章）
  - 每年12月左右发布，覆盖全国492场赛事的 **A类/B类/C类** 认证
  - 这是赛事级别（A/B/C）的 **唯一权威来源**
  - 查询方法：搜"2026年马拉松赛事目录 田协" 或 "2026福建省马拉松赛事完整名单"
  - 注意：A类≠世界田联标牌！A类是田协认证，世界田联标牌（白金标/金标/精英标/标牌）是另一个体系

### 第二梯队 - 世界田联标牌等级
- **世界田联官网** 或 相关报道（如搜狐、腾讯新闻等）
  - 查询方法：搜"2026世界田联标牌 中国 马拉松"
  - 等级从高到低：**白金标 > 金标 > 精英标 > 标牌**
  - 2026年中国有：白金标2场（厦门、上海）、金标9场（无锡、扬州、兰州、太原、北京、东营、杭州、深圳、广州）、精英标8场、标牌10场

### 第三梯队 - 赛事详情（日期、报名、起终点、爬升等）
- **闹跑 nowrun.cn** — 数据最全的第三方聚合站
  - 首页提取492场赛事的名称和ID（href="/race/XXX"）
  - 详情页：https://www.nowrun.cn/race/{ID} 包含日期、起跑时间、地点、报名时间线、赛事设项（规模+报名费）、起终点、累计爬升、平均海拔、官方公众号
  - 注意：闹跑的"A类/B类/C类"是前端显示的田协认证，但很多赛事详情页可能不显示，不要单独依赖
- **去跑马 gorunma.cn** — 替补数据源（日期、地点、级别补充）
- **中国田协官网 mls.chinaath.com** — 备用数据源

## 关键数据规则

### 认证级别双体系
1. **田协认证（A/B/C类）** — 从田协官方目录获取（第一梯队）
2. **世界田联标牌（白金标/金标/精英标/标牌）** — 从世界田联获取（第二梯队）

### 福建省2026赛事真实分级（已验证的权威数据）
```
A类（9场）：厦门马拉松、福州马拉松、福建永定土楼半程、仙游马拉松、武夷山马拉松、厦门海沧半程、厦门环东半程、莆田马拉松、宁德马拉松
B类（1场）：泰宁半程马拉松
C类（7场）：石狮半程、马尾琅岐半程、光泽半程、霞浦(三沙)马拉松、漳州半程、泉州晋江马拉松、漳州华安土楼半程
```

### 位置推荐逻辑
1. 世界田联标牌等级优先 → 田协A/B/C类 → 距用户距离
2. 具体分级推荐策略：
   - **白金标/金标/精英标** — 必跑级别，无论远近
   - **田协A类** — 品质不错，结合地理位置综合判断
   - **标牌级/田协B类** — 可考虑（如离得近）
   - **C类** — 除非极近（30min车程内）或完赛物资特别好，一般不推荐
3. 用户IP位置通过上下文获取（`<system-reminder>` 中的用户信息），不要写死在skill里
