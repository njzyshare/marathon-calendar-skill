# 马拉松个人赛历和推荐

## 工作流程

### 1. 用户提问 → 先回答
- **默认只输出对话框文字结论**，不要自动生成文件
- 文字结论需包含：比赛日期、报名状态、认证级别、交通耗时、赛道备注
- 对话结束后**主动询问**是否需要整理成报告

### 2. 报告 → 询问格式
生成报告前必须问用户：
- 是否生成？（默认不）
- 如果要，选 **HTML页面** 或 **图片长图（PNG）**
- 直接生成，不追问工具

### 3. 数据采集
从闹跑 nowrun.cn 主页提取全部赛事（href="/race/XXX"），再逐个抓详情页。
替补：gorunma.cn / mls.chinaath.com

### 4. 认证查询
- 田协 A/B/C 类：搜"2026年马拉松赛事目录 田协"
- 世界田联标牌：搜"2026世界田联标牌 中国 马拉松"

### 5. 交通查询
必须从 12306/携程 查真实数据，禁止估算。
搜格式："出发城市到目标城市 高铁 最快"

### 6. 赛道备注
每条推荐必须附带备注：爬升数据、景点、特色。
搜索引擎搜 "赛事名 赛道 爬升"
